#Demo Project

Requirements
- Create a GUI to accept device inputs and its asscociated interfaces with validations, save it to
database and display device results in a grid view. Display the device interfaces
in a grid view inside a dialog box which will open on click of “View Interfaces” button using.
- Device Inputs:
- hostname - unique
- loopback0_ipv4 – unique ipv4 address
- Interface Inputs: (Each device can have multiple interfaces)
- interface – unique for device, valid interface name e.g. Loopback100, Loopback150,
Loopback200
- interface_ipv4 – unique ipv4 address

Commands to Run,Build the project are mentioned in sequential manner.
- ng serve 
- ng build
